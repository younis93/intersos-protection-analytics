INTERSOS Protection Analytics - Portable Windows Edition

1. Unzip the package to a local folder.
2. Double-click INTERSOS Protection Analytics.exe.
3. The dashboard opens directly in a native Windows application window; no browser or console window is used.
4. Upload an approved Excel workbook with the sheets: Assessments, Legal Services, and Deportation.

Privacy
This package contains no workbook or operational case data. Uploaded workbooks are processed only on this computer in memory and are cleared when the application window is closed. The application listens only on 127.0.0.1 and is not accessible from other devices on the network.

To end the session
Close the native application window. This stops the local service and clears the current uploaded workbook.
